## 这是什么

我是肥总，一个重度龙虾玩家。折腾龙虾终端的过程中，我积累了一些渠道和资源——**现货信息、内部折扣、token 福利、各地 POC 政策与补贴、工作流模板**——与其自己藏着，不如做一个 skill 直接共享出来。

这不是一个官方客服机器人，也不是一个套着代码壳的广告页。

就是一个龙虾玩家，把自己踩过的坑和搞到的资源整理成了一个 skill。

## 能帮你干什么

### 🔧 工具层：终端选型助手

告诉 skill 你的使用场景，它帮你匹配最合适的龙虾终端配置：

```
场景输入示例：
- "本地跑 7B 模型 + 日常开发"
- "多 Agent 工作流 + 内容生产"
- "预算 8000 以内，主力开发机"
```

输出包括推荐配置、预估性能、部署建议，以及——如果当前有现货的话——直达购买链接。

### 🏛️ POC 政策直通车：先试再买，还能拿补贴

很多人不知道，各地政府和联想都有针对 AI 终端 / 智算设备的 **POC（概念验证）试用政策**，符合条件的企业、团队甚至个人开发者可以申请：

- **免费或低成本试用** 龙虾终端一段时间
- **地方政府补贴**（部分地区对 AI 算力 / 智算终端采购有专项补贴）
- **企业采购绿色通道**（批量采购走 POC 流程可以大幅降低决策风险）

这些信息以前都散落在各地销售体系里，普通用户根本不知道怎么申请。我花了不少时间把它整理在一起了。

| 北京 | AI 算力补贴 + POC | 注册在京企业 | 算力采购补贴最高 XX% | [`poc/beijing.md`](poc/beijing.md) |

#### POC 申请流程（通用版）

```
第 1 步：确认资格
│  查看 poc/ 目录下你所在城市的政策文档
│  确认你的企业 / 团队是否符合申请条件
│  （个人开发者？看下方"个人开发者特别通道"）
│
第 2 步：准备材料
│  基础材料：营业执照 / 团队证明 + 使用场景说明
│  不需要写 PPT，一段话讲清楚你要用龙虾终端干什么就行
│  └── 场景说明模板见 poc/application-template.md
│
第 3 步：提交申请
│  方式 A：通过本 skill 直接提交（优先处理）
│  　　　　python poc_apply.py --city beijing --scenario "本地Agent开发"
│  方式 B：在 Issue 中发起 POC 申请（打上 poc-request 标签）
│  方式 C：按各地文档中的官方渠道自行申请
│
第 4 步：审核与对接
│  通过 skill 提交的申请，我会帮你对接到对应区域的负责人
│  通常 3-5 个工作日内回复
│
第 5 步：设备交付 & 试用
   试用期内有任何问题，Issue 里随时找我
```

#### 🚀 个人开发者特别通道

严格来说，大部分 POC 政策面向企业客户。但如果你是：

- 有明确 AI 项目的独立开发者
- 在做开源项目且有一定 Star 量的开发者
- 自由职业的 AI 内容创作者 / 自动化工程师

我可以帮你以「个人工作室」的方式走通道申请。不保证 100% 成功，但我会尽力帮你争取。在 Issue 里简单说明你的情况就行。

#### ⚡ 龙虾先锋 × POC 优先权

通过本 skill 提交的 POC 申请享有以下优先权益：

| 优先项 | 说明 |
|-------|------|
| 优先审核 | 不用走常规排队，我帮你直接对接负责人 |
| 材料简化 | 不需要走标准企业采购流程，场景说明即可 |
| 补贴指导 | 我帮你确认你所在城市是否有叠加的政府补贴 |
| 试用延长 | 试用期内如果需要延期，我帮你协调 |
| 转采优惠 | 试用后决定购买，叠加 skill 专属折扣码 |

> 本质上就是：**把 POC 资源和各地政府补贴信息，用极客能理解的方式透明化，帮你减少中间的扯皮环节。**

## FAQ

**Q：这是联想官方项目吗？**

不是。这是我个人维护的项目。我在联想工作，日常接触得到一些内部渠道和资源信息，所以能拿到普通用户拿不到的东西。但这个项目的运作方式就是一个个人开源项目——我整理信息，你直接用，没有中间商。

**Q：福利是真的吗？折扣码会不会是骗邮箱的？**

折扣码直接用，不需要填任何个人信息。领了不用也不会怎样。Token 福利随机器激活自动到账。你可以翻翻 Issue 里其他用户的反馈。

**Q：我不买龙虾终端，这个 skill 对我有用吗？**

工作流模板部分不限设备，本地大模型部署和 Agent 开发的方案在其他硬件上也能参考。但福利和 POC 部分确实是龙虾终端专属。

**Q：POC 试用真的免费吗？有什么限制？**

全国通用的联想企业 POC 试用是免费的，试用期通常 15-30 天，到期后可以选择购买（叠加 skill 折扣）或归还。地方政府补贴部分各地政策不同，有的是直接补贴采购款，有的是算力券，具体看 `poc/` 目录下你所在城市的文档。

**Q：我是个人开发者，不是企业，能申请 POC 吗？**

严格来说大部分 POC 面向企业客户，但如果你有明确的 AI 项目或者有一定影响力的开源项目，我可以帮你走内部通道试试。在 Issue 里说明你的情况就行，我会尽力协调。

**Q：POC 政策信息多久更新一次？**

各地政策我会按季度集中更新一轮。如果某个城市发布了新的 AI 产业补贴政策，我看到后会第一时间更新。你也可以在 Issue 里提供你知道的政策线索，帮大家一起受益。

**Q：会一直更新吗？**

只要我还在这个岗位上，每周至少更新一次。如果哪天不更新了，我会在这里说明原因。
